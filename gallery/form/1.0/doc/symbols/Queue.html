Queue ()

描述：文件上传队列

继承自： Base

源码： base.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • Queue. duration {} <static> 动画速度
  • Queue. event {} <static> 支持的事件
  • Queue. files {} <static> 文件信息数据
  • Queue. status {} <static> 文件的状态
  • Queue. statusConfig {} <static> 状态类配置，queue和file参数会被组件内部覆盖，传递无效
  • Queue. target {} <static> 队列元素
  • Queue. tpl {} <static> 模板

Methods

  • add (files, callback) 向上传队列添加文件
  • clear () 清理队列
  • fileStatus (index, status, args) 获取或设置文件状态
  • getFile (index) 获取指定索引值的队列中的文件
  • getFileIndex (fileId) 根据文件id来查找文件在队列中的索引
  • getFiles (status) 获取指定状态下的文件
  • getIndexs (type) 获取等指定状态的文件对应的文件数组index的数组 param {String} type 状态类型
  • remove (indexOrFileId, callback) 删除队列中指定id的文件
  • render () 运行组件
  • restore (需要恢复的数据) 将数据恢复到队列中
  • updateFile (index, data) 更新文件数据对象，你可以追加数据
