Theme ()

描述：上传组件主题基类

继承自： Base

源码： theme.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • auth {Auth} Auth（上传验证）实例
  • button {Button} Button 按钮实例（_init()下并不存在）
  • cssUrl {String} css模块路径
  • duration {Number} 动画速度
  • fileTpl {String} 队列使用的模板
  • name {String} 主题名
  • oPlugin {Array} 插件类集合
  • plugins {Array} 需要加载的插件，需要手动实例化
  • queue {Queue} Queue（上传队列）实例
  • queueTarget {String} 队列目标元素（一般是ul），队列的实例化过程在Theme中
  • uploader {Uploader} Uploader 上传组件实例

Static Properties

Methods

  • afterUploaderRender (uploader) 在上传组件运行完毕后执行的方法（对上传组件所有的控制都应该在这个函数内）
  • displayFile (isShow, target, callback) 控制文件对应的li元素的显影
