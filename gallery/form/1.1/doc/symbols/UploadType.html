UploadType (config)

描述：上传方式类的基类，定义通用的事件和方法，一般不直接监听此类的事件

继承自： Base

源码： base.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • action {String} 服务器端路径
  • data {Object} 传送给服务器端的参数集合（会被转成hidden元素post到服务器端）

Static Properties

  • UploadType.event {} 事件列表

Methods

  • stop () 停止上传
  • upload () 上传文件

Events

  • error () 上传失败后触发
  • start () 开始上传后触发
  • stop () 停止上传后触发
  • success () 上传成功后触发
