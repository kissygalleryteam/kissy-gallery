Uploader (config)

描述：异步文件上传组件，支持ajax、flash、iframe三种方案

继承自： Base

源码： base.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • autoUpload {Boolean} 是否自动上传
  • button {Button} Button按钮的实例
  • curUploadIndex {Number} 当前上传的文件对应的在数组内的索引值，如果没有文件正在上传，值为空
  • isAllowUpload {Boolean} 是否允许上传文件
  • queue {Queue} Queue队列的实例
  • restoreHook {String} 已经存在的文件数据待提取的容器钩子
  • serverConfig {Object} 服务器端配置。action：服务器处理上传的路径；data： post给服务器的参数，通常需要传递用户名、token等信息
  • type {String|Array} 采用的上传方案，当值是数组时，比如“type” : ["flash","ajax","iframe"]，按顺序获取浏览器支持的方式，该配置会优先使用flash上传方式，如果浏览器不支持flash，会降级为ajax，如果还不支持ajax，会降级为iframe；当值是字符串时，比如“type” : “ajax”，表示只使用ajax上传方式。这种方式比较极端，在不支持ajax上传方式的浏览器会不可用；当“type” : “auto”，auto是一种特例，等价于["ajax","iframe"]。
  • uploadFilesStatus {String} 存在批量上传文件时，指定的文件状态
  • uploadType {UploaderType} 上传方式实例
  • urlsInput {UrlsInput} UrlsInput实例
  • urlsInputName {String} 存储文件路径的隐藏域的name名

Static Properties

  • Uploader.event {} 组件支持的事件列表，{ RENDER:'render', SELECT:'select', START:'start', PROGRESS : 'progress', COMPLETE:'complete', SUCCESS:'success', UPLOAD_FILES:'uploadFiles', CANCEL:'cancel', ERROR:'error' }
  • Uploader.status {} 文件上传所有的状态，{ WAITING : 'waiting', START : 'start', PROGRESS : 'progress', SUCCESS : 'success', CANCEL : 'cancel', ERROR : 'error', RESTORE: 'restore' }
  • Uploader.type {} 上传方式，{AUTO:'auto', IFRAME:'iframe', AJAX:'ajax', FLASH:'flash'}

Methods

  • cancel (index) 取消文件上传，当index参数不存在时取消当前正在上传的文件的上传。cancel并不会停止其他文件的上传（对应方法是stop）
  • getUploadType (type) 获取上传方式类（共有iframe、ajax、flash三种方式）
  • isSupportAjax () 是否支持ajax方案上传
  • isSupportFlash () 是否支持flash方案上传
  • render () 运行组件，实例化类后必须调用render()才真正运行组件逻辑
  • restore (data) 添加默认数据到队列，不带参数的情况下，抓取restoreHook容器内的数据，添加到队列内
  • stop () 停止上传动作
  • upload (index) 上传指定队列索引的文件
  • uploadFiles (status) 批量上传队列中的指定状态下的文件

Events

  • cancel () 取消上传后触发
  • complete () 上传完成（在上传成功或上传失败后都会触发）
  • error () 上传失败后触发
  • progress () 正在上传中时触发，这个事件在iframe上传方式中不存在
  • select () 选择完文件后触发
  • start () 开始上传后触发
  • success () 上传成功后触发
  • uploadFiles () 批量上传结束后触发

Class Detail

  • Uploader (config)
    异步文件上传组件，支持ajax、flash、iframe三种方案
    参数：
    •   config {Object} 组件配置（下面的参数为配置项，配置会写入属性，详细的配置说明请看属性部分）
    •   config.button {Button} *，Button按钮的实例
    •   config.queue {Queue} *，Queue队列的实例
    •   config.type {String|Array} *，采用的上传方案
    •   config.serverConfig {Object} *，服务器端配置
    •   config.urlsInputName {String} *，存储文件路径的隐藏域的name名
    •   config.isAllowUpload {Boolean} 是否允许上传文件
    •   config.autoUpload {Boolean} 是否自动上传
    依赖：
    • UrlsInput
    • IframeType
    • AjaxType
    示例：
    var uploader = new Uploader({button:button,queue:queue,serverConfig:{action:'test.php'}})
