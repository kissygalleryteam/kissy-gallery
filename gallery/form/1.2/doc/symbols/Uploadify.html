Uploadify ()

描述：改造自jquery的异步上传插件uploadify

继承自： Theme

源码： index.js

author：明河

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • cssUrl {String} css模块路径
  • elCount {KISSY.NodeList} 统计上传张数的容器
  • fileTpl {String} 队列使用的模板
  • name {String} 主题名（文件名），此名称跟样式息息相关
  • plugins {Array} 需要加载的插件，需要手动实例化
  • Fields borrowed from class Theme: auth, button, duration, oPlugin, queue, queueTarget, uploader

Methods

  • afterUploaderRender (uploader) 在上传组件运行完毕后执行的方法（对上传组件所有的控制都应该在这个函数内）
  • getFilesLen (status) 获取成功上传的图片张数，不传参的情况获取成功上传的张数
  • Methods borrowed from class Theme: displayFile, render
