Auth (uploader, config)

描述：文件上传验证，可以从按钮的data-auth伪属性抓取规则配置

继承自： Base

源码： base.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • rules {Object} 上传验证规则，每个规则都是一个数组，数组第一个值为规则，第二个值为错误消息
  • uploader {Uploader} 上传组件实例

Methods

  • getRule (ruleName) 获取指定规则
  • isUploaderType (type) 判断上传方式
  • testAll () 验证上传数、是否必须上传
  • testAllowExt (file) 测试是否是允许的文件上传类型
  • testMax () 检验是否达到最大允许上传数
  • testMaxSize (file) 检验是否超过允许最大文件大小，留意iframe上传方式此验证无效
  • testRepeat (file) 检验文件是否重复（检验文件名，很有可能存在误差，比如不同目录下的相同文件名会被判定为同一文件）
  • testRequire () 检验是否必须上传一个文件

Events

  • error () 当验证出错时触发
