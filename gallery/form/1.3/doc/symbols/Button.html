Button (target, config)

描述：文件上传按钮，ajax和iframe上传方式使用

继承自： Base

源码： base.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • cls {Object} 样式
  • disabled {Boolean} 是否可用,false为可用
  • fileInput {KISSY.Node} 对应的表单上传域
  • inputContainer {KISSY.Node} 文件上传域容器
  • multiple {Boolean} 是否开启多选支持，多选目前有兼容性问题，建议禁用
  • name {String} 隐藏的表单上传域的name值
  • target {KISSY.Node} 按钮目标元素
  • tpl {String} 隐藏的表单上传域的模板

Methods

  • hide () 隐藏按钮
  • render () 运行
  • reset () 重置按钮
  • show () 显示按钮

Properties Detail

  • cls {Object} — 样式
  • disabled {Boolean} — 是否可用,false为可用
    default：false
  • fileInput {KISSY.Node} — 对应的表单上传域
    default：""
  • inputContainer {KISSY.Node} — 文件上传域容器
    default：""
  • multiple {Boolean} — 是否开启多选支持，多选目前有兼容性问题，建议禁用
    default：true
  • name {String} — 隐藏的表单上传域的name值
    default："fileInput"
  • target {KISSY.Node} — 按钮目标元素
    default：null
  • tpl {String} — 隐藏的表单上传域的模板
