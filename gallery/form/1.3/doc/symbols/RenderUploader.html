RenderUploader (buttonTarget, queueTarget, config)

描述：异步文件上传入口文件，会从按钮的data-config='{}' 伪属性中抓取组件配置

版本： 1.2

源码： index.js

Config Attributes（Properties）- 使用get和set方法获取/设置属性

  • authConfig {Object} 验证配置
  • button {Button} Button（上传按钮）的实例
  • buttonTarget {String|HTMLElement|KISSY.NodeList} 按钮目标元素
  • queue {Queue} Queue（上传队列）的实例
  • queueTarget {String|HTMLElement|KISSY.NodeList} 队列目标元素
  • theme {String} 主题引用路径，当值为""时，不使用uploader主题。非内置主题，值为模块路径，比如"refund/rfUploader"
  • themeConfig {Object} 主题配置，会覆盖data-theme-config中的配置，不再推荐使用伪属性的方式配置主题参数
  • uploader {Uploader} 上传组件实例
  • uploaderConfig {Object} 上传组件配置

Events

  • init () 上传组件完全初始化成功后触发，对uploader的操作务必先监听init事件

Class Detail

  • RenderUploader (buttonTarget, queueTarget, config)
    异步文件上传入口文件，会从按钮的data-config='{}' 伪属性中抓取组件配置
    params：
    •   buttonTarget {String | HTMLElement} *，上传按钮目标元素
    •   queueTarget {String | HTMLElement} 文件队列目标元素，再不需要显示文件信息的情况下这个参数可以设置为null
    •   config {Object} 配置，该配置会覆盖data-config伪属性中的数据
    require：
    • Uploader
    • Auth
    example：
    
     选择要上传的文件
     
     
      

     
    KISSY.use('gallery/form/1.3/uploader/index', function (S, RenderUploader) {
     var ru = new RenderUploader('#J_UploaderBtn', '#J_UploaderQueue');
     ru.on("init", function (ev) {
        var uploader = ev.uploader;
     })
})

Properties Detail

  • authConfig {Object} — 验证配置
    since：1.2
    example：
                                                                        
             //验证配置
             authConfig: {
                 allowExts:[
                     {desc:"JPG,JPEG,PNG,GIF,BMP", ext:"*.jpg;*.jpeg;*.png;*.gif;*.bmp"},
                     '不支持{ext}格式的文件上传！'
                 ],
                 max:[3, '每次最多上传{max}个文件！']
             }
  • button {Button} — Button（上传按钮）的实例
    default：""
  • buttonTarget {String|HTMLElement|KISSY.NodeList} — 按钮目标元素
    default：""
  • queue {Queue} — Queue（上传队列）的实例
    default：""
  • queueTarget {String|HTMLElement|KISSY.NodeList} — 队列目标元素
    default：""
  • theme {String} — 主题引用路径，当值为""时，不使用uploader主题。非内置主题，值为模块路径，比如"refund/rfUploader"
    default：“default”
  • themeConfig {Object} — 主题配置，会覆盖data-theme-config中的配置，不再推荐使用伪属性的方式配置主题参数
    since：1.2
    example：
                                                                        
 //配置主题样式路径
themeConfig:{
    cssUrl:'gallery/form/1.3/uploader/themes/default/style.css'
}
  • uploader {Uploader} — 上传组件实例
    default：""
  • uploaderConfig {Object} — 上传组件配置

Events Detail

    •   ev.queue {Queue} Queue的实例
