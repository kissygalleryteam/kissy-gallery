当前位置： KISSY Gallery Huabao API 
作者：法海 <fahai@taobao.com>
功能：huabao
描述：此类是一个包装类，对下引用着 model 中的 poster 和 pic，对上提供业务方面的高层接口
　　　Pic 类主要定义了一些标准的属性字段，并且提供一个可被重写的 parse 方法。
　　　Huabao 类从 Base 继承而来，使用标准的 get / set 方法来访问字段。
源码：huabao.js

API

属性

  • {Object} dataSource 数据源
  • {Object} data 画报相关数据，格式为 
    {
    poster: new Poster(),
    pics: [new Pic(), ...]
}
  • {Object} rawData 以 json 形式存放的画报相关数据，只读
  • {Boolean} autoRendering 如果为 true，则使用默认的模板渲染画报数据，默认 false
  • {HTMLElement} srcNode 画报的容器节点
  • {HTMLElement} nav 缩略图导航区的容器节点，关于 content 和 nav 一块的设计参见 switchable
  • {HTMLElement} content 大图区的容器节点
  • {HTMLCollection} triggers 缩略图导航区的所有孩子节点集合
  • {HTMLCollection} panels 大图区的所有孩子节点集合
  • {String} navCls
  • {String} contentCls
  • {String} triggerCls
  • {String} panelCls
  • {Number} activeIndex 当前选定的图片索引
  • {Number} length 图片列表长度，即 huabao.get("data").pics.length
  • {Number} pageIndex 为 carousel 形式的缩略图区服务。当前缩略图区的页码
  • {Number} pageSize 为 carousel 形式的缩略图区服务。缩略图区一页的长度，一般和 carousel 的 steps 一致
  • {Number} pageLength 为 carousel 形式的缩略图区服务。整个 carousel 一共有多少页
