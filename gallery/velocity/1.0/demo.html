velocity

try it online注：此页面不支持ie访问。 使用需配合velocity命令行工具(on github)做线下编译，直接使用velocity可能存在性能问题，velocity本身逻辑比较 复杂。

API

构造器

new Velocity(asts)

参数

  • {Object} [ asts = {} ] asts是velocity/parse.js执行parse方法后得到的语法树，同时可以在线下使用velocity --build *.vm就行编译得到。

实例方法

(new Velocity(asts)).render(context);
  • {Object} [ context = {} ] 数据对象，velocity执行时从context中获取值，context的属性可以是方法。
